Role Name
=========

# common setup centos7

```
* disable selinux
* disable firewalld
* setup sysstat

* set timezone, locale
  localectl set-timezone
  localectl set-locale

* package install:
      - "bash-completion"
      - "git"
      - "bind-utils"
      - "sysstat"
      - "dstat"
      - "nc"

* history setting
  /etc/profile.d/history.sh
```

Requirements
------------
None

Role Variables
--------------

```
locale: ja_JP.UTF-8
zone: Asia/Tokyo
```


Dependencies
------------
None

Example Playbook
----------------

    - hosts: servers
      vars:
        zone: UTC
      roles:
         - ansible-common-centos7
